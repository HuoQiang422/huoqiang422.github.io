# 📚 Hexo博客内容更新指南

欢迎学习如何在你的Hexo博客中添加和更新内容！

## 🎯 创建新文章的方法

### 方法一：使用Hexo命令（推荐）
```bash
# 创建新文章
npx hexo new "文章标题"

# 创建新页面
npx hexo new page "页面名称"

# 创建草稿
npx hexo new draft "草稿标题"
```

### 方法二：手动创建
直接在 `source/_posts/` 目录下创建 `.md` 文件

## 📝 文章Front Matter配置

```yaml
---
title: 文章标题                    # 必需
date: 2025-08-27 10:00:00         # 发布日期
updated: 2025-08-27 12:00:00      # 更新日期（可选）
tags:                             # 标签
  - 标签1
  - 标签2
categories:                       # 分类
  - 主分类
  - 子分类
cover: 图片URL                     # 封面图片
description: 文章描述              # 文章摘要
top: true                         # 置顶（可选）
---
```

## 🔄 更新流程

### 1. 编辑内容
- 修改现有文章或创建新文章
- 使用Markdown语法编写内容

### 2. 本地预览
```bash
# 清理缓存
npx hexo clean

# 生成静态文件
npx hexo generate

# 启动本地服务器
npx hexo server
```

### 3. 访问预览
打开浏览器访问：`http://localhost:4000`

## 📊 内容管理建议

1. **定期备份**：定期备份 `source` 目录
2. **图片管理**：建议使用图床服务存储图片
3. **SEO优化**：合理设置标题、描述和标签
4. **分类规划**：建立清晰的分类体系

## 🚀 部署更新

```bash
# 生成并部署
npx hexo generate --deploy

# 或分步执行
npx hexo generate
npx hexo deploy
```
