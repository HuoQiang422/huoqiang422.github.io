## 🎨 Butterfly主题特色

Butterfly是一个功能丰富、美观大方的Hexo主题，具有以下特点：

### ✨ 主要特性

- **响应式设计** - 完美适配各种设备

- **代码高亮** - 支持多种编程语言

```python
# Python示例
def fibonacci(n):
    """生成斐波那契数列"""
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# 输出前10个斐波那契数
for i in range(10):
    print(f"F({i}) = {fibonacci(i)}")
```
